# Lincare Saúde - Assistência Domiciliar de Qualidade

Lincare Saúde é um website moderno e completo para uma empresa de assistência à saúde domiciliar, desenvolvido com Next.js. O site oferece informações sobre serviços de saúde personalizados no conforto do lar, com foco em cuidado humanizado e profissionais qualificados.

## 🚀 Tecnologias Utilizadas

- **Next.js 16** - Framework React para aplicações web
- **React 19.2.0** - Biblioteca JavaScript para interfaces de usuário
- **TypeScript** - Superset do JavaScript com tipagem estática
- **Tailwind CSS** - Framework CSS utilitário para estilização
- **Lucide React** - Biblioteca de ícones
- **Radix UI** - Componentes acessíveis e desacoplados
- **Vercel Analytics** - Solução de analytics

## ✨ Características

- Design responsivo adaptado para todos os dispositivos
- Componentes modulares e reutilizáveis
- Formulário de contato funcional com validação
- Navegação intuitiva com âncoras
- Layout moderno e profissional
- Foco em acessibilidade e experiência do usuário
- Integração com chamadas telefônicas e WhatsApp

## 📋 Serviços Oferecidos

- **Cuidados Pós-Operatórios** - Acompanhamento especializado após cirurgias
- **Assistência ao Idoso** - Cuidados personalizados com atenção à mobilidade
- **Administração de Medicamentos** - Controle seguro e conforme prescrição médica
- **Reabilitação Física** - Programas de fisioterapia domiciliar
- **Cuidados Paliativos** - Atendimento compassivo focado no conforto
- **Monitoramento de Saúde** - Acompanhamento contínuo de sinais vitais

## 🛠️ Instalação e Execução

### Pré-requisitos

- Node.js (versão 18 ou superior)
- npm, pnpm ou yarn

### Instalação

1. Clone este repositório:
   ```bash
   git clone <URL_DO_REPOSITORIO>
   cd lincaresaude
   ```

2. Instale as dependências:
   ```bash
   # Usando pnpm (como indicado pelo pnpm-lock.yaml)
   pnpm install
   
   # Ou usando npm
   npm install
   
   # Ou usando yarn
   yarn install
   ```

3. Execute o servidor de desenvolvimento:
   ```bash
   # Usando pnpm
   pnpm dev
   
   # Ou usando npm
   npm run dev
   
   # Ou usando yarn
   yarn dev
   ```

4. Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.
